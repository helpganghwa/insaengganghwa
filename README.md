# 인생강화 — insaengganghwa

시간기반 idle + 한국식 RPG 강화 모바일 웹 게임.

## 기술 스택

Next.js 16 · React 19.2 · TypeScript · Tailwind v4 · Bun · Supabase · Drizzle · 포트원

## 시작하기

```bash
bun install
bun dev
```

http://localhost:5174

프로덕션 도메인: **insaengganghwa.com** (Vercel, 추후 연결)
