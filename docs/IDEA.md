# 인생강화 (insaengganghwa) — 컨셉

### 한 줄 컨셉

**시간기반 idle + 한국식 RPG 강화** — 강화 1회에 수 초~수 시간이 걸리고, 그 사이 게임을 떠나 있어도 진행된다. 돌아와서 결과를 확인하고 "+99 떴다"고 카톡에 자랑한다.

### 장비

- 슬롯: 무기 / 방어구 / 장신구 — 3슬롯 고정 (4+ 미도입)
- **부위마다 강화 타이머 별도 동작** — 무기 강화 중에도 방어구/장신구는 동시 강화 가능. 부위당 2슬롯(총 6개 동시 진행)
- 자랑 단위 2종 — **장비 단위**(장비 1개 상세) 또는 **장비 전체**(3슬롯 세트 + 내 프로필)

### 강화

- **인스턴스 강화 아님** — 강화 시도마다 시간이 소요됨 (1회 수 초 ~ 수 시간; 고레벨 도달은 누적 수 주)
- **시간 곡선은 완만** — 기하급수 ❌, idle 친화적 곡선 (모디파이어로 추가 단축 가능)
- **강화 단계 무제한** — 단, 시간/자원 비용은 단계마다 증가
- **+51까지 안전** — 실패해도 유지 (하락 없음)
- **+52부터 하락 구간** — 실패 시 단계 하락 가능. 파괴 개념 없음, 경고 모달 없음
- **+100부터** — 강화 시도마다 같은 카탈로그 아이템 1개 제물 소모 (시간 + 제물)
- 보호권/축복권 개념 없음
- **소요 시간 비례 강화확률** — base rate × (경과/총시간). 끝까지 기다리면 공시 최대치, 일찍 시도하면 낮은 확률 도박
- **강화 시도는 무료** — 강화석 개념 없음, 언제든 강화 시도 가능. 필요 없는 장비는 분해하면 보석 고정 지급

### 재화

- **보석으로 강화 타이머 단축** — 1보석 ≈ 분 단위 단축 (런칭 후 데이터 보고 리밸런싱)
- 보석 획득: IAP, 일일 미션, 레이드 보상, **보급 상자 개봉 시 보석 드롭** (확률). 보상형 광고는 v1 미도입(BALANCE §6.4)

### 운영

- **서버** — dev, master-dev, master 분기. 실운영 전까지 dev, master-dev만 사용. DB 분기 X
- **도메인** — dev는 로컬, master-dev는 Vercel 배포, master는 추후 실운영 도메인 연결

### 레이드

- **보석으로 레이드 오픈** — 오픈 비용 1,000 다이아 (즉시 차감, 환불 없음)
- **슬롯** — 유저당 동시 진행 3개 · 일일 5회 (호스팅+참여 합산)
- **공유** — 레이드 생성 후 공유링크 생성, 카톡방에 링크로 참여 가능
- **보상** — 1회 이상 공격한 참여자 **전원 동일** (기여도 가중 없음). 기본 참가 100다이아 + 페이즈 돌파마다 추첨(50% 100다이아 / 50% 보급 상자, 슬롯 1/3)
- **보스** — 죽이는 개념보다 데미지 누적, phase 1~N 피통 깎는 형식. 깎은 페이즈 많을수록 보상 증가
- **이미지** — Pixellab에서 보스 이미지 생성 (애니메이션 추가)
- **보스 종류** — 슬라임킹, 오크족장, 드래곤(서양식), 돌골렘, 타락천사
- **스토리** — 레이드 참가 팝업에 보스 판타지 스토리 추가
- **시간** — 레이드 오픈 후 6시간 동안 공격 가능, 참여자당 기본 10회 + 다이아로 추가 공격(점진 증가). 미스 없음, 크리 ×1.5, 데미지 ±30%

### 도감

- **프로필에 도감** — 미획득 disable된 이미지, 획득 시 활성화. 최대 강화 수치 표기

### 랭킹

- **최고 강화** — 가장 메인이 되는 기준
- **합산 강화** — 도감 최고 강화 수치 합
- **전투력** — 장착 장비의 전투력 합

### 장비 추가 계획

- 등급/특이 "히든" 티어 없음 — 일반 **카탈로그 아이템(슬롯별)**을 지속 추가해 도감·외관 다양성 확장
- 추가 아이템도 성능 동일 — 강화·초월로만 성장, 차이는 외관/도감뿐
