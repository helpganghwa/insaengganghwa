# GCRB 등급분류 신청폼 — 입력 내용 백업

게임콘텐츠등급분류위원회(PC/온라인 게임물) 신규접수 신청폼에 입력한(또는 입력할) 값 백업.
플랫폼: 웹/PWA → **PC·온라인 게임물**. 신청번호(app): 183254.

---

## 1단계. 신청서 작성 — 게임물 기본정보

| 항목 | 값 |
|---|---|
| 게임물 제목 | 인생강화 |
| 플랫폼 | PC/온라인 (웹·PWA) |
| 서비스 형태 | 웹 브라우저 기반 온라인 게임 |
| 희망 등급 | **12세이용가** |

### 게임물 개요 (요약)

> 인생강화는 장비를 시간에 걸쳐 강화하고 캐릭터를 육성하는 한국식 RPG 웹게임입니다.
> 플레이어는 무기·방어구·장신구를 강화·수집하고, 픽셀아트 아바타를 꾸미며, 레이드·대난투 등
> 협동·경쟁 콘텐츠에 참여합니다. 시간 기반 성장과 픽셀아트 그래픽이 특징입니다.

> 참고: 플레이어는 이름(닉네임)을 정하지 않아도 진행 가능합니다.
