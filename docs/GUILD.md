# GUILD — 길드 (협력 성장 · 지역 점령)

> 유저가 길드를 결성해 **함께 길드를 키우고, 월드맵의 지역을 공격·수비해 점령**하는 사회·경쟁 콘텐츠. 강화/초월로 쌓은 개인 전투력이 길드 전투력으로 환원되고, 점령은 서버 결정론 정산으로 판정되므로 실시간 PvP 인프라·치트 위험 없이 1인 운영에 친화적이다. 게임 전 시스템을 묶는 **엔드게임 리텐션 축**.

관련: `GDD.md`(코어 루프) · `BALANCE.md §3`(전투력) · `MELEE.md`(결정론 정산 모델 선례) · `SCHEMA.md`(스키마) · `MAIL.md`(경고·보상 우편) · `PROFILE.md`(Pixellab 에셋 생성 파이프라인 — 문양 프리셋)

## 0. 설계 원칙 (협상 불가)

- **결정론·비동기 (대난투 모델)**: 실시간 공성 없음. 점령은 정해진 KST 시각에 길드 전투력 스냅샷 → 결정론 정산. 안티치트·1인 운영 친화.
- **등급 없음·공정성 유지**: 기부·과금이 **개인 전투력으로 환산되지 않는다**. 돈으로 점령력/길드 진척을 독식하는 P2W 금지(즉시 정체성 훼손). 길드 수익화는 **외형·집단 편의** 한정.
- **반-스노볼**: 1등 길드 영구 독식 방지. 개인 일일 기부 상한·길드 수용 상한·점령 혜택 상한으로 격차 누적을 구조적으로 억제.
- **idle 저압력 유지**: 미션은 "매일 새로 하라"가 아니라 **하던 행동의 자동 집계**. 기여 의무가 숙제가 되지 않게.
- **출시 시퀀싱**: 길드는 **마지막 콘텐츠**. 인구가 있어야 재밌으므로 정식 오픈 후 **DAU 임계(길드가 빠르게 차는 수준) 확인 뒤 투입**. 콜드스타트 단계에 넣으면 "빈 길드 = 죽은 게임" 신호가 되어 역효과.

> 모든 수치는 **시작 테스트값**이며 `lib/game/balance.ts` 단일 출처로 이관 + `tests/balance/` 시뮬레이션으로 튜닝한다(CLAUDE §3.5).

## 1. 길드 결성 / 가입 / 탈퇴

- **결성 비용**: **10,000💎**. 결성자는 길드장. 개설은 소수(길드당 1명)·가입은 다수 무료라 높은 비용이 전체 진입장벽이 아니며, **즉흥 개설로 인한 빈 길드 양산을 억제**하고 다이아 sink로 기능한다(앵커 1,440💎/일 대비 ≈ 7일치).
  - (선택·TBD) 비-다이아 자격 게이트 병행 검토 — 예: 계정 생성 N일 경과 + 최소 합산 전투력. 진지한 길드장 필터를 다이아 액수 인상 없이 강화.
- **길드명**: 자유 텍스트(예 2~12자). **사전 AI 검수 없음** — 부적절명은 **신고 기반 사후 대응**(운영 강제 개명/해산)으로 처리. 중복은 허용 여부 TBD.
- **수용 인원**: `15 + (길드레벨 − 1)`, **상한 30**(레벨 16에서 도달). §2 곡선과 맞물림.
- **길드 아이덴티티** → §1.5.
- **탈퇴 후 24시간 재가입 불가**: 용병성 길드 호핑·정찰 어뷰징 방지. 서버 타임스탬프 비교(결정론·구현 단순).
  - 적용 범위: 탈퇴 후 24h간 **다른 길드 가입 불가**.
  - ⚠ 월드맵 막판 갈아타기(정산 직전 승자 길드로 점프)는 24h 잠금만으로 부족 — **점령 보상 자격을 시즌/정산 시점에 묶는 규칙**으로 별도 보강(§5 TBD).

## 1.5 길드 아이덴티티 (이름 · 문양 · 공지)

- **이름**: §1 — 자유 텍스트, 사전 검수 없음, 신고 기반 사후 대응.
- **문양(엠블럼)**: **Pixellab AI로 미리 생성한 프리셋 N종 중 선택**(자유 업로드·실시간 생성 없음).
  - **왜**: 업로드/실시간 생성은 NSFW·저작권 모더레이션과 스토리지 부담이 큼. **빌드타임 프리셋**이면 모더레이션 표면 0 + 픽셀아트 톤 일관 + 즉시 렌더.
  - 에셋 생성은 Pixellab 호출(유료) → **생성 전 사용자 확인 필수**. 초기 프리셋 수·테마(방패/깃발/원형 등 형태 × 색)는 TBD.
  - **BM 연결(§6)**: 프리미엄·한정 문양(점령 기념 등)은 비-P2W 외형 매출 후보.
- **공지**: 짧은 자유 텍스트(예 ≤60자). **편집 권한 = 길드장/부길드장만**. **사전 AI 검수 없음** — 멤버 전용 노출 + 신고 기반 사후 대응. idle 특성상 자유 채팅(§7 보류) 대신 v1의 1줄 소통 채널.

> 사전 검수를 두지 않으므로 **이름·공지에 대한 신고 경로와 운영 처리(개명/해산/제재) 동선**은 반드시 마련한다(사후 모더레이션 의존).

## 2. 길드 성장 (기부 · 미션 → 길드 XP → 레벨)

### 2.1 기부 (일 3회, KST 자정 리셋)

| 기부 | 비용 | 길드 XP | 개인 기여도 |
|---|---|---|---|
| 1회차 | 무료 | 10 | 10 |
| 2회차 | 50💎 | 30 | 30 |
| 3회차 | 100💎 | 70 | 70 |

- 일 최대 **110 길드 XP / 150💎** per 멤버.
- **왜 체증 + 상한**: 고래도 하루 150💎로 동일 상한 → 돈으로 길드 진척 독식 불가(반-스노볼). 무료 1회는 캐주얼의 무압력 데일리 탭.
- **왜 효율 0.6→0.7 XP/💎**: 100💎(70XP)가 50💎(30XP)보다 다이아당 효율이 살짝 높아, **3회차까지 채울 동기** 부여(무료만 찍고 마는 것 방지).
- 경제 정합: 앵커 **1,440💎=하루** 대비 일 기부 150💎 ≈ **10%** — 부담 없는 sink.

### 2.2 미션 (자동 집계형 — 하던 행동이 쌓임)

- **일일**: 강화 시도 · 보급 개봉 · 대난투 참가 등 합계 **~50 길드 XP/일** (세부 항목 TBD).
- **주간**: 누적 목표 **~250 길드 XP/주**.
- 활성 멤버 1인 ≈ 하루 **150~200 길드 XP** 기여(기부+미션 전부 채울 때).

### 2.3 길드 레벨 곡선 (앞 싸게 = 수용 빨리 열림 / 뒤 비싸게 = 엔드게임 sink)

| 구간 | 레벨당 필요 XP | 수용 인원 |
|---|---|---|
| L1 → L5 | ~500 | 15 → 20 |
| L6 → L10 | ~1,500 | 20 → 25 |
| L11 → L16 | ~4,000 | 25 → **30(상한)** |
| L17+ | (체증) | 30 고정 — 보상은 **버프·월드맵 혜택**으로 전환 |

- 누적 약 **34,000 XP → 활성 길드 기준 약 1개월에 수용 상한 30 도달**(길드가 커질수록 가속).
- 레벨 16 이후는 인원 대신 **길드 버프 / 월드맵 점령력·슬롯** 등으로 계속 성장(엔드게임).

## 3. 개인 기여도

- **누적 기여도 = 본인이 생성한 길드 XP와 1:1 누적**(기부 + 미션). 한 숫자로 투명 관리.
- 용도: ① **무임승차 판단**(길드 운영), ② **길드장 자동 위임 tie-break**(§4), ③ (TBD) 월드맵 보상 분배·주간 집계.

## 4. 직책 / 운영

- **직책**: 길드장 · 부길드장(복수 가능) · 멤버. (권한 세부 TBD)
- **길드장 7일 미접속 → 자동 위임**:

  - 부길드장 전원 잠수/부재 → **동일 규칙으로 다음 활성 멤버 연쇄 승격**(빈 길드 영구 잠금 방지).

- 결정론(서버 타임스탬프) — 수동 개입 0.

## 5. 월드맵 — 지역 공격 / 수비  *(상세 TBD)*

> 월드맵에서 지역을 공격·수비해 점령하는 길드 경쟁. **상세 기획 예정.** 확정 시 본 절을 채운다.

### 5.1 지역 (레이드 5지역 모티브) — 확정

월드맵은 **레이드 지역을 모티브**로 한다(아트·로어·테마 재사용 → 신규 정체성 에셋 부담 0, 레이드↔월드맵 세계관 일관). 5지역 + 방위 배치:

1. **드래곤 화산** — 서(西)
2. **잊힌 신전** — 북(北)
3. **슬라임 늪** — 남(南)
4. **타락한 천사들의 하늘섬** — 중앙 (대륙 심연 위 부유섬, 부서진 신성 유적·천사상)
5. **오크 부락** — 동(東)

- 레이드 보스(slime_king·orc_chief·stone_golem·dragon_west·fallen_angel)와의 1:1 매핑은 테마적이며 엄밀 대응은 TBD.

### 5.2 점령 단위 — 확정: 지역당 20구역 × 5 = 100구역

- **(A) 다중 거점 방식**: 각 지역이 **20개 구역(노드)**으로 분할, 월드맵 전체 **100구역**. 여러 길드가 나눠 점령 → 희소성 해소(100노드면 다수 길드 수용).
- 구역 단위로 **공격/수비** 진행. 구역별 소유 길드·점령 상태를 보유.

### 5.3 지도 표현 (아이소메트릭 베이스맵 + 코드 노드 오버레이)

- **베이스맵**: 단일 아이소메트릭 픽셀아트 월드맵 1장 — `public/sprites/guild/worldmap.png`(1024², `image-rendering: pixelated`로 렌더). 5지역이 하나의 링형 대륙으로 자연스럽게 이어지고, **중앙에 타락한 천사들의 부유섬이 대륙보다 높게 떠 있음**. 어두운 톤·지역간 고저차·완만한 경사 경계.
- **제작 파이프라인**: 정밀 베이스를 외부 이미지 모델로 생성 → sharp로 워터마크 제거 + 팔레트 양자화(또렷한 픽셀 + 용량 최적). 지역마다 **작은 오브젝트(유적·캠프·용암구·슬라임풀)를 여러 규모로 분산** → 노드 거점 후보.
- **노드(100구역)**: 이미지에 굽지 않고 **코드 오버레이**(클릭 가능 + 소유 길드 색). 베이스맵은 배경만 담당.

### 5.4 점령전 (Conquest) — 확정 골격

- **점령 단위**: 구역(§5.2, 100구역). 길드가 구역 단위로 점령·소유.
- **공격/방어**: 중립·적 구역은 **공격**, 자기 점령 구역은 **방어** 가능.
- **전투 방식**: **대난투와 동일한 결정론 시뮬레이션**(서버가 전 전투 산출·리플레이, 실시간 PvP 아님). 배치된 인원·전투력 기반.
- **전투 시각**: 매일 **KST 12:00~13:00** 1회. 12:00 배치 잠금 → 결정론 산출 → 결과·리플레이 공개·소유권 갱신.
- **배치(그 외 시간)**: 전투 시각 전까지 멤버를 **공격 인원**(목표 구역 지정) 또는 **방어 인원**(자기 점령 구역에만) 배치.
- **안개(은닉)**: 공격·방어 인원은 **자기 길드 것만 열람**. 적의 배치 규모·구성은 전투 결과 전까지 비공개(맵의 구역 소유 현황 자체는 공개).

**전제**(§0 연장, 유지):
- 결정론·비동기 / 반-스노볼(점령 혜택 상한) / 시간대 공정(KST 고정) / 공·수 비대칭(수비 프리미엄) / 무과금 공정(점령력 유료구매 금지) / 막판 갈아타기 차단(보상 자격 게이팅 §1).

**결정 대기(TBD)**:
- ① 한 구역을 **여러 길드가 동시 공격** 허용 여부(자유경쟁 vs 1:1 매칭).
- ② **인접 규칙** — 점령 구역에 인접한 곳만 공격 가능(확장형) vs 어디든 공격.
- ③ **승패 판정식** — 대난투식 시뮬의 구체 산식(전투력 합/시뮬), **수비 프리미엄** 수치.
- ④ **멤버 배치 제한** — 1인 1배치/일? 구역당 인원 상한?
- ⑤ **점령 보상·혜택** — 구역 보유 시 얻는 것(일일 재화·외형·버프), 반-스노볼 상한.
- ⑥ 무방비(방어 인원 0) 점령 구역·중립 구역 처리.

## 7. 출시·운영 리스크 메모
