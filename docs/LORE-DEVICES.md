# 로어 장치 대장 (LORE-DEVICES.md)

> 150종 = 150개의 서로 다른 '이야기 장치'. 한 번 쓴 장치는 영구 폐기.
> 새 로어는 여기 없는 장치만. 발상이 닮으면 탈락(레지스터 우연 일치는 허용).
> 형식: `key — 장치 한 줄`
