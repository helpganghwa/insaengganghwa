## 1. 세계관 · 줄거리(시나리오) 요약

- 별도의 서사형 스토리 진행(퀘스트 라인)이 없는 **육성·수집 중심의 롤플레잉** 게임이다.
- 플레이어는 별도의 초기 설정 없이(닉네임은 선택 사항) 바로 시작하며, **무기·방어구·장신구** 3종 장비를 수집·강화하여
  자신의 전투력을 무한히 끌어올리는 것이 목표다.
- 세계관은 검·마법의 판타지를 배경으로 하며, 장비 아이템·레이드 보스마다 짧은 판타지 설정
  글(로어)이 붙어 분위기를 형성한다. 폭력적·선정적 서사는 포함하지 않는다.
- 핵심 정서는 "오래 기다려 강화에 성공하고 그 성과를 자랑한다"는 **성취·수집·경쟁**이다.

## 2. 조작 방법

- **모바일·PC 웹 브라우저에서 터치/클릭으로만 조작**한다. 별도 설치 파일·전용 컨트롤러·단축키
  (Hotkey)는 없다.
- 화면 하단 탭 내비게이션으로 각 기능(강화소·보급소·레이드·랭킹·상점·프로필·설정)에 이동한다.
- 모든 상호작용은 버튼 탭 기반이며, 실시간 액션 조작(이동·조준 등)은 없다.

---

## 3. 핵심 게임 진행(플레이 루프)

1. 로그인 후 진행 중이던 **강화 결과 확인**(접속하지 않은 동안 서버 시계로 진행됨).
2. **일일 보급 상자**를 수령·개봉하여 장비를 획득한다.
3. 장비를 **강화 큐에 등록**한다(1회당 수 초~수 시간 소요, 게임을 떠나도 타이머 진행).
4. 성과(고강화·초월)를 **카카오톡으로 자랑(공유)**한다.
5. 여유가 되면 **레이드·대난투·길드 점령전** 등 경쟁/협동 콘텐츠에 참여한다.

- "짧은 접속(30초~2분)으로 결과 확인·다음 강화 등록"을 하루 여러 번 반복하는 방치형(idle)
  루프가 게임의 뼈대다. 시간 압박·만료가 없어 언제 복귀해도 결과가 보존된다.

## 4. 주요 시스템

### 4.1 장비 · 성장(조합) 시스템

- 장비는 **무기 / 방어구 / 장신구 3슬롯**뿐이며, 등급·희소성·부가옵션 체계가 없다.
- 장비의 강함은 오직 **두 축**에서 나온다.
  1. **강화(+레벨)** — 시간 기반 강화로 상승(아래 §4.2).
  2. **초월(Transcend)** — 같은 아이템의 중복 수집이 쌓이면 자동으로 상승(아래 §4.3).
- **아이템 조합 시스템 = 강화 + 초월.** 별도의 재료 합성/제작/분해는 없으며, 같은 카탈로그
  아이템의 중복이 곧 성장 재료(초월 진행도)로 자동 흡수된다.
- **총 전투력** = 보유한 모든 카탈로그 아이템(중복 제외)의 개별 전투력 합. 착용은 외형(프로필)
  전용이며 전투력에는 영향이 없다.

### 4.2 강화 시스템 (시간 기반, 확률형)

게임의 핵심 시스템이다.

- **비용은 시간뿐** — 강화 시도에 재화·재료가 들지 않는다. 강화 레벨이 높을수록 1회 소요
  시간이 길어진다(수 초~수 시간). 게임을 종료해도 서버 시계로 진행된다.
- **확률 결과(확률형 항목)** — 완료 시점에 서버에서 난수로 결과가 결정된다.
  - 결과: **성공(단계 상승) / 유지(변화 없음) / 하락(단계 하락)** (일부 구간은 2단계 상승 "메가"
    포함). 정확한 확률 구성비는 게임 내 **확률 공시 페이지(`/probability`)** 에 전부 표기되며
    코드값과 1:1로 일치한다.
  - **시간 비례 성공률** — 끝까지 기다리면 공시된 기준 확률에 도달하고, 일찍 시도하면 확률이
    낮아진다.
- **파괴·소멸 없음** — 강화 실패의 최악 결과는 "한 단계 하락"이며, 장비가 사라지는 개념은 없다.
  낮은 구간(+51 이하)은 하락조차 없이 "유지"만 발생한다.
- **유료 결제와의 관계** — 강화 자체는 무료다. 유료재화(다이아)는 오직 **대기 시간을 단축**하는
  용도로만 쓰인다(도박적 재화 투입이 아니라 시간 절약).
- **모든 난수는 서버에서만** 결정되며 클라이언트는 결과 연출만 재생한다(조작 불가).

### 4.3 초월 시스템 (자동, 수집 기반)

- 보급으로 **이미 보유한 아이템**이 다시 나오면 그 아이템의 초월 진행도가 자동으로 +1 누적되고,
  임계치에 도달하면 **초월 레벨이 자동 상승**한다. 유저의 선택·확정 조작이 없다(무RNG·즉시).
- 상한이 없으며 강화와 독립적으로 전투력에 배수로 작용한다.

### 4.4 보급 시스템 (확률형 아이템 / 뽑기)

- **보급 상자(무기·방어구·장신구 3종)** 를 열면 해당 슬롯의 아이템 중 하나가 나온다.
- **확률 구조** — 각 슬롯 상자는 그 슬롯의 활성 아이템 전체가 **균등 확률(1 ÷ 아이템 수)** 로
  나온다. 등급·천장(pity) 개념이 없다. **각 아이템의 이름과 당첨 확률을 확률 공시 페이지에
  전부 표기**한다.
- 미보유 아이템은 신규 획득(도감 해금), 보유 아이템은 초월 진행도로 자동 누적된다.
- 보급 상자 획득 경로: 일일 무료 보급, 출석 보상, 레이드·대난투·점령전 보상, 유료 상점 패키지.

### 4.5 레이드 (플레이어 호스팅 협동)

- 호스트가 유료재화(다이아 300)로 개설하고, 카카오톡 공유 링크로 최대 10명이 무료 참여하는
  **협동 보스 콘텐츠**다. 6시간 동안 누적 데미지로 보스 페이즈를 깎는다.
- 보스는 판타지 몬스터 5종(슬라임킹·오크족장·돌골렘·드래곤·타락천사)이며, 전투는 **데미지
  수치 누적** 방식이다. 유혈·잔혹 묘사는 없다.
- 보상은 참여자 전원 동일(보급 상자). 확률형 요소는 지급되는 보급 상자 개봉(위 §4.4)에 한한다.

### 4.6 대난투 (일일 자동 경쟁, 결정론)

- 강화 1회 이상 경험한 유저가 **매일 자동 참가**하는 결정론 방식의 다인 경쟁 콘텐츠다. 서버가 전투력을 기반
  으로 결과를 자동 산출하고, 유저는 결과·리플레이를 관람한다.
- **무료 고정** — 참가·순위에 재화를 투입하는 경로가 전혀 없다(유료 우위 없음). 따라서 확률형·
  사행성 요소가 아니다.
- 전투 표현은 픽셀아트 캐릭터 간 데미지 수치 교환이며, 잔혹·선정 표현이 없다.

### 4.7 길드 · 점령전

- 유저는 길드에 가입해 협동할 수 있고, 매일 정해진 시각 구역을 두고 **길드 간 점령전**(결정론
  시뮬레이션)을 벌인다. 각자 공격/방어에 참여를 등록하고 서버가 결과를 산출한다. 유료 우위·
  확률형 요소가 없다.

### 4.8 랭킹 · 도감

- 최고 강화·전투력·합산 강화 기준의 전역 누적 랭킹(시즌 리셋 없음)과, 수집한 아이템을 기록하는
  도감을 제공한다.

### 4.9 프로필 아바타 (AI 자동 검수)

- 유저는 자신의 장비를 반영한 캐릭터 아바타(픽셀아트)를 생성할 수 있다. 생성된 이미지는
  **AI 비전(선정성·부적절성 자동 검수)** 을 통과해야만 적용되며, 부적절 이미지는 자동 반려된다.
  또한 신고·운영자 조치 체계를 갖춰 청소년 유해·선정 표현을 차단한다.

## 5. 주요 아이템 (아이템 리스트 · 조합 시스템)

- 아이템은 **무기·방어구·장신구 슬롯별 카탈로그**로 구성된다(예: 검·도끼·완드·활·지팡이 등
  무기류 / 갑주·로브·드레스 등 방어구류 / 투구·관·안경·부채 등 장신구류). 각 아이템은 하나의
  슬롯에 속하며, **아이템 간 성능 차등이 없다**(외형·수집·초월 식별 단위).
- **전체 아이템 목록과 슬롯 구분은 게임 내 도감 및 보급 확률 공시 페이지에서 확인 가능**하며,
  심의용 계정으로 전 아이템을 열람할 수 있다.
- **아이템 조합/성장 = 강화(시간) + 초월(중복 수집 자동 누적)** 두 축뿐이며(§4.1~4.3), 별도의
  재료 합성·제작·분해·거래는 존재하지 않는다(현금 거래소·RMT 없음).

  붙는다. 유혈·공포 표현은 배제한다.

## 7. 확률형 아이템 확률 공시 (게임산업법 제33조)

- **대상**: (1) 강화(성공/유지/하락 확률), (2) 보급 상자(슬롯별 균등 확률).
- **표시 위치**: 게임 내 하단 "확률 공시" 링크 및 공개 웹페이지 `https://ganghwa.app/probability`.
  강화 페이지·보급 페이지 하단에서 해당 섹션으로 바로 이동한다.
- **표시 내용**: 강화 4결과의 확률 구성비·위험 구간, 보급 상자별 **모든 아이템의 이름과 당첨
  확률(1/N)** 을 전량 표기한다.
- **정합성**: 표시값은 게임 로직 상수(`lib/game/balance.ts`, 보급은 활성 아이템 실시간 조회)에서
  직접 산출되어 **실제 게임 판정과 항상 동일**하다. 확률 변경 시 사전 공지 및 이력 보존 정책을
  둔다.

## 8. 요금(결제) 체계

- **유료재화**: 다이아. 카카오페이·토스·신용카드 등으로 결제(PG: 포트원)하여 충전한다.
- **판매 품목**: 다이아 패키지 및 보급 상자 패키지(상점). 다이아 용도는 (1) 강화 대기시간 단축,
  (2) 레이드 개설/추가 공격 등이다.
- **환금성 없음** — 유료로 얻는 것은 모두 게임 내 재화·아이템이며, 현금 환전·유저 간 현금 거래·
  거래소가 없다(사행성 낮음의 핵심 근거).
- **결제 영수증**: 전자상거래법에 따라 결제 시 이메일로 영수증을 발송한다.
- **청약철회·환불**: 미사용 유료재화는 관련 법령에 따른 청약철회·환불 정책을 약관(환불정책)에
  명시하고 제공한다.

## 9. 청소년 보호

- **본인인증(KMC/PASS)** — 유료 결제 진입 시 본인인증을 요구한다(무료 콘텐츠는 인증 없이 이용).
- **미성년자 결제 한도** — 미성년자 월 결제 한도(7만원)를 적용하여 초과 결제를 차단하는 로직을
  구현한다. 생년 정보는 해시·검증값 위주로 최소 저장한다.
- **미성년자 환불 보호** — 관련 법령에 따른 미성년자 결제 취소·환불 보호를 제공한다.
- **약관·정책 게시** — 이용약관·개인정보처리방침·청소년보호정책·환불정책을 게임 내(설정)와
  웹페이지(`/legal/*`)에 게시한다.

## 10. 콘텐츠 등급 관련 자기 평가 (희망등급: 12세이용가)

| 평가 요소 | 내용 | 정도 |
|-----------|------|------|
| 선정성 | 아바타는 AI 선정성 자동 검수 + 신고·조치 체계. 노출·성적 표현 없음 | 없음 |
| 폭력성 | 전투는 픽셀아트 캐릭터 간 **데미지 수치 누적**. 유혈·잔혹·신체훼손 묘사 없음 | 낮음 |
| 공포 | 판타지 몬스터 설정글 수준. 공포 연출 없음 | 없음 |
| 언어 | 시스템 텍스트 위주. 자유 채팅 미도입(길드 공지 1줄 등 제한적) | 낮음 |
| 범죄·약물 | 해당 없음 | 없음 |
| **사행성** | 강화·보급에 **확률형 요소 + 유료재화**가 있으나 **획득물이 게임 내 재화·아이템이고 현금 환전·거래소가 없다.** 확률은 전량 공시. 대난투·점령전 등 경쟁 콘텐츠는 무과금·결정론 | 낮음~보통 |

- 확률형 요소와 유료 결제가 존재하므로 전체이용가는 부적합하나, **폭력·선정·공포 표현이 없고
  획득물의 현금 환전·거래가 없으며 확률을 전량 공시**하므로 **12세이용가**를 희망한다(최종
  등급은 심의 결과에 따른다).
