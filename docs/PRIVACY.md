[상호](이하 "회사")는 「개인정보 보호법」 등 관련 법령을 준수하며, 이용자의 개인정보를 다음과 같이
처리합니다.

## 1. 수집하는 개인정보 항목 및 방법

| 구분 | 항목 | 수집 방법 |
|------|------|-----------|
| 계정(소셜 로그인) | 카카오 회원번호(고유 식별자), [프로필 닉네임·이메일 — 동의 범위에 한함] | 카카오 로그인 |
| 본인확인(필요 시) | 이름, 생년월일, 성별, 내외국인 여부, 통신사, 중복가입확인정보(CI), 연계정보(DI) | 본인인증기관(PASS/KMC) |
| 결제 | 결제 일시·금액·상품, 결제수단 구분, 거래 식별자(결제수단 상세는 결제대행사가 처리) | 유료 결제 시 |
| 이메일(영수증) | 이메일 주소 | 영수증 발송 시 |
| 알림(선택) | 푸시 구독 정보(endpoint·키), 알림 수신 설정 | 푸시 동의 시 |
| 자동 수집 | 접속 IP, 쿠키·로컬스토리지, 기기·브라우저 정보, 서비스 이용·접속 로그 | 서비스 이용 과정에서 자동 |
| 게임 데이터 | 닉네임, 캐릭터·아이템·재화·활동 기록 | 서비스 이용 과정에서 생성 |

- 회사는 민감정보를 원칙적으로 수집하지 않으며, 본인확인 결과는 **성인 여부·인증 일시·출생연도 해시 등
  최소 식별값** 위주로 보관합니다.

## 2. 개인정보의 처리 목적
1. 회원 식별·인증, 가입 및 계정 관리
2. 게임 서비스 제공·운영, 데이터 저장·동기화
3. 유료서비스 결제·정산·영수증 발급·환불 처리
4. 연령 확인 및 청소년 보호, 미성년 결제 보호
5. 부정이용·어뷰징 방지 및 서비스 안정성 확보
6. 고객 문의 응대, 공지·알림 발송(동의 범위)
7. 서비스 개선을 위한 통계·분석(식별 최소화)

## 3. 보유 및 이용 기간
- 회원 탈퇴 시 지체 없이 파기함을 원칙으로 합니다. 단, 다음은 관련 법령에 따라 보관합니다.
  - 계약·청약철회·대금결제·재화공급 기록: 5년 (전자상거래법)
  - 소비자 불만·분쟁처리 기록: 3년 (전자상거래법)
  - 표시·광고 기록: 6개월 (전자상거래법)
  - 접속(로그) 기록: 3개월 (통신비밀보호법)
- 부정이용 방지를 위해 제재 이력 등은 [기간] 동안 별도 보관할 수 있습니다.

## 4. 개인정보의 제3자 제공
- 회사는 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만 법령에 근거가 있거나 수사기관의 적법한
  요청이 있는 경우는 예외로 합니다.

## 5. 개인정보 처리의 위탁
회사는 서비스 제공을 위해 아래와 같이 개인정보 처리를 위탁합니다.

| 수탁자 | 위탁 업무 |
|--------|-----------|
| Supabase | 데이터베이스·인증 호스팅 |
| Vercel | 애플리케이션 인프라·호스팅 |
| 카카오 | 소셜 로그인 인증 |
| [포트원] | 결제 처리 대행 |
| [PASS/KMC] | 본인확인 |
| Resend | 이메일(영수증) 발송 |
| Upstash | 이용 제한(어뷰징 방지) 처리 |
| Anthropic | 이용자 생성 콘텐츠(프로필) 검수 |
| [PostHog / Sentry] | 서비스 분석·오류 모니터링 |

- 위탁계약 시 개인정보의 안전한 관리를 위한 사항을 규정하며, 수탁자·업무 변경 시 본 방침을 통해 고지합니다.

## 6. 개인정보의 국외 이전
일부 수탁자는 국외에 소재하여 개인정보가 국외로 이전·처리될 수 있습니다.

| 이전받는 자 | 국가 | 이전 항목 | 이전 시점·방법 | 목적 |
|-------------|------|-----------|----------------|------|
| Supabase | [미국 등] (데이터센터 [서울]) | 계정·게임 데이터 | 서비스 이용 시 네트워크 전송 | DB·인증 호스팅 |
| Vercel | 미국 (함수 리전 [서울]) | 접속·이용 정보 | 서비스 이용 시 네트워크 전송 | 인프라·호스팅 |
| Resend | 미국 | 이메일 주소 | 영수증 발송 시 | 이메일 발송 |

| Upstash | [국외] | 이용자 식별자 | 서비스 이용 시 | 어뷰징 방지 |

- 이용자는 국외 이전을 거부할 수 있으나, 거부 시 서비스 이용이 제한될 수 있습니다.
